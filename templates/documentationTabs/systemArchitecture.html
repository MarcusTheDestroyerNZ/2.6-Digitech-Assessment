Explaination

In this code I have a function called save_player_data() that saves the player's data into a JSON file.
This saves all important player information, such as money, research, power, upgrades, and the state of the game world (like placed blocks and unlocked areas) into a file for loading next time the game is opened.
It writes this data to playerData.json using the json.dump().
I use the now.strftime() method to get the current date and time when the player logs out, which is also saved in the JSON file. I use this to track when the player last played the game, giving offline progress when they return.
